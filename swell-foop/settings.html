Game settings

You can change the game look according to your preference.

Change the size

You can set the size of the board according to your preference:

  1. Select Swell Foop ▸ Preferences.

  2. The default board size is Small. You can change it from Board size under Setup.

Change the number of objects

You can increase the number of different objects in the game.

  1. Select Swell Foop ▸ Preferences.

  2. The default number of colors is 3, you can increase or decrease it from Number of colors under Setup. The maximum number of colors allowed are 4.

Change the look

To change the appearance of the objects:

  1. Select Swell Foop ▸ Preferences.

  2. The default theme is Shapes and Colors. You can select another theme from Theme under Appearance.

Change the speed

The default animation is designed to be smooth and graceful. If you like playing fast and furious then this may be too slow for you. You can speed up the game play at the expense of animation quality.

  1. Select Swell Foop ▸ Preferences.

  2. To speed up the game, check the Zealous Animation under Operation.
