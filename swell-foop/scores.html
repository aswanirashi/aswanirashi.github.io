Scores

You can checkout your current game scores at the top of the game screen. To view scores obtained in previous games:

  1. Select Swell Foop ▸ Scores.

  2. The dialog by default displays scores for 6 x 5, 3 colors game. To view others, select the size from the menu whose scores you wish to see.
