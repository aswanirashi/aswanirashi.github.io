Start a new game

You can start playing a new game, save the current game and continue playing it after sometime. To start a new game:

  1. Click the New Game button on the toolbar.

  2. A dialog box appears which lets you save the current game to play it later or to simply leave the game. To save the current game, and continue it later, select Save game for later. To leave the game without saving, select Abandon game.
