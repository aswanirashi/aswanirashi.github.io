Chess engines

GNOME Chess supports any Chess Engine Communication Protocol or Universal Chess Interface compatible chess engines. You can edit the gnome-chess/data/engines.conf for the desired engine.

  1. To edit the file for just yourself, copy it from $(datadir)/gnome-chess/engines.conf to $XDG_CONFIG_HOME/gnome-chess/engines.conf. These will usually be /etc/gnome-chess/engines.conf and ~/.config/gnome-chess/engines.conf, but some distros are different.

  2. To edit for all users using root privledges, you can edit $(datadir)/gnome-chess/engines.conf directly.

  3. You can use the delay-before-move option to select how long Chess should wait before reporting the human's move to the engine. The default value of delay is 2 seconds.

  4. Chess can pass options to a UCI engine each time the engine is told to move. To know which options it supports, you can check the engine's documentation.

  5. The other options get passed to the chess engine when the game starts.

  6. You can also pass different options to the engine binary depending on difficulty level, see the GNU Chess and Phalanx configurations to know more.
