Start a new game

Klotski offers a large collection of games, to play a new game:

  1. Select Klotski ▸ New Game. Alternatively, you can click on Start Over. It will start a new game of the default puzzle, Only 18 Steps.

Play a different puzzle

  1. To play a different puzzle, click on View Puzzles.

  2. Choose a game category and double click to start playing.
