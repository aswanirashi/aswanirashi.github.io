Players settings

You can change the number of Human Players and Computer Opponents.

  1. Select Tali ▸ Preferences.

  2. You can increase or decrease the number of Human Players from Number of players. The maximum number of players can be six. You can change their names as well from Player Names.

  3. You can change the number of Computer Opponents from Number of opponents and set the difficulty level of the opponent from the drop down menu next to Difficulty.

Enable delay between the rolls

You can enable the delay between the rolls by checking the Delay between rolls.
